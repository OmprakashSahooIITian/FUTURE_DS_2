# Facebook Ads Performance Intelligence Dashboard

## Project Overview
This project focuses on analyzing Facebook Ads campaign data to evaluate overall campaign performance, user engagement, Click-Through Rate (CTR), conversions, and Return on Investment (ROI).  
The objective is to transform raw advertising data into actionable insights using an interactive Power BI dashboard that supports data-driven marketing decisions.

---

## Objectives
- Measure campaign performance across key marketing KPIs  
- Analyze advertising spend efficiency and ROI  
- Track CTR, impressions, clicks, and conversions  
- Compare performance across campaigns, demographics, and interests  
- Enable campaign optimization through data visualization  

---

## Tools & Technologies
- Power BI – Data modeling, DAX measures, dashboard development  
- Excel / CSV – Data source and preprocessing  
- Marketing Analytics – CTR, CPC, ROI, conversion analysis  
- Dashboard Storytelling – KPI cards, trend analysis, and comparative visuals  

---

## Dataset Description
The dataset consists of Facebook advertising campaign data with the following attributes:

- campaign_id  
- ad_id  
- start_date  
- end_date  
- age  
- gender  
- interest1  
- interest2  
- interest3  
- impressions  
- clicks  
- spent  
- total_conversions  
- approved_conversions  

---

## Data Cleaning & Preparation
The following steps were performed to prepare the dataset for analysis:

- Converted date columns to Date format  
- Set numeric fields (impressions, clicks, spend, conversions) to appropriate numeric data types  
- Handled missing and invalid values  
- Standardized categorical fields such as age group, gender, and campaign ID  
- Created calculated measures using DAX  

---

## Key Metrics (KPIs)
The dashboard tracks the following key performance indicators:

- Total Spend  
- Total Impressions  
- Click-Through Rate (CTR)  
- Total Conversions  
- Cost per Conversion  
- Revenue vs Total Spend  
- Return on Investment (ROI)  

---

## Dashboard Features
- KPI cards for high-level performance monitoring  
- Campaign-wise CTR comparison  
- Click distribution by campaign  
- Spend versus ROI analysis  
- Time-series analysis of spend and clicks  
- Interactive slicers for age group, gender, campaign ID, and target interests  

---

## Insights Generated
- Identification of high-performing campaigns with strong CTR and ROI  
- Detection of inefficient spend areas for optimization  
- Analysis of engagement trends over time  
- Performance comparison across demographic segments and interests  

---

## Business Value
This dashboard enables marketing teams to:
- Optimize advertising budgets  
- Improve audience targeting strategies  
- Reduce cost per conversion  
- Increase overall campaign ROI
